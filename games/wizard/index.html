I wanted to make a game where you went around as a wizard zapping people. I did make that I suppose, but I wanted there to be more of an actual game. As I got more and more into it, I realized the only fun part was making sprites. So, this is just a bunch of sprites running around. You play as the wizard from the end of this video: https://youtu.be/9Lr3h2jeMD4

Sorry if the audio is a little loud.


Controls:                   Debug Controls:
Movement - WASD                   f1 - ugly debug menu
Jump - Space                   R - Restart Game
Shoot- Left Click                   F - Die
f10 - fullscreen (can bug out)                   f2 - revive
